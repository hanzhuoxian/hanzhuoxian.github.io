## hexo 使用

1. 安装 hexo

   ```bash
   npm install -g hexo-cli
   ```

2. 下载依赖

    ```bash
    yarn install
    ```

3. 启动本地服务

    ```bash
    hexo server
    hexo s # 简写
    ```

4. 生成静态文件

    ```bash
    hexo generate
    hexo g # 简写
    ```
5. 创建文章

    ```
    hexo new -p path title
    ```
